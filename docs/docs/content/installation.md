# :octicons-download-16: Installation

Install the RayforceDB VS Code Extension to get started with developing RayforceDB applications right inside VS Code!

!!! note ""
    Simply search for "RayforceDB" in VS Code / Cursor marketplace

[Open VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=RayforceDB.rayforce-vscode){ .md-button }
[Open OpenVSX Registry](https://open-vsx.org/extension/RayforceDB/rayforce-vscode){ .md-button }


## Command Line

Install directly from the terminal using the VS Code CLI:

```bash
code --install-extension RayforceDB.rayforce-vscode
# OR
cursor --install-extension RayforceDB.rayforce-vscode
```
